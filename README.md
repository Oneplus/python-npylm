# NPYLM

[ベイズ階層言語モデルによる教師なし形態素解析](http://chasen.org/~daiti-m/paper/nl190segment.pdf)のC++実装です。

現在ビタビアルゴリズムによる単語分割を実装しています。

もうしばらくお待ちください。

## 動作環境

- Boost

## ビルド

```
make install
```

## 実行

```
python train.py -f textfile.txt
```

## 注意事項

研究以外の用途には使用できません。
